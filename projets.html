Culture Quest

Description du projet :

Culture Quest est un RPG rogue like tour par tour à la japonaise, il suit une histoire soigneusement fabriquée (en s'inspirant de J-RPG cultes) et un système de combat digne des plus grands qui stimule les connaissances culturelles. Enchainez les combats et arrivez au boss final pour SAUVER la PLANÈTE avec votre CULTURE ! Tout cela codé en IJava !

Compétences mise en œuvre :

    Savoir Faire :
  • Compétences algorithmiques
  • Compréhension de IJava
  • Diviser pour régner
  • Gestion de fichiers et organisation
    Savoir Être :
  • Savoir communiquer
  • Voir les difficultés de chacun pour favoriser l’entraide
  • Savoir être patient et persévérant pour trouver la meilleure solution au problème, ne pas être hâtif

Applications d'appariements étudiants

Description du projet :

Ce projet avait pour but de créer une application qui permet de lier des adolescents participant à un séjour linguistique dans un appariement optimal à l’aide d’in score affinité. Pour cela, nous avons donc développé une application en Java en suivant un cahier des charges, nous avons mis en place et étudié des graphes pour l’appariement optimal et enfin, nous avons mis toute une interface fonctionnelle avec du JavaFX.

Compétences mise en œuvre :

    Savoir Faire :
  • Maîtriser le langage Java (types, fonctions, packages, ouvrir/fermer fichiers...)
  • Maîtriser les théories des IHM (Bertin, Fitts...) pour créer une interface
  • Maîtriser les théories des graphes (Chemin optimal, pondération…)
    Savoir Être :
  • Savoir communiquer
  • Voir les difficultés de chacun pour favoriser l’entraide
  • Savoir consulter et comprendre la documentation Java
  • Savoir répartir les tâches entre les membres de l’équipe

Site Écomobilité Playstation

Description du projet :

Ce projet a pour objectif de mobiliser les compétences de développement web pour pouvoir réaliser un site web pour une filiale fictive d’éco-mobilité pour Playstation, comprenant un système de connexion, de paiement, de choix de covoiturage et de location.
